# AkzoNobel Agent Bricks Workshop

## Welcome

Build AI agents on Databricks over synthetic AkzoNobel coatings data. You will call
AI straight from SQL, create no-code Agent Bricks agents (Genie, Knowledge Assistant,
extraction, classification), evaluate agent quality with MLflow, and give an agent
short-term memory. No prior agent code required.

> All data is synthetic. Product names, customers, suppliers, and documents are invented for the workshop.

## Before you start

- Use **Chrome**
- Do **not** use Incognito / InPrivate
- Allow pop-ups and cookies

## How to begin

1. Launch the lab.
2. Open **`_AGENDA/00 - Start Here`**.
3. Run **`01 - Setup Check`** first — it confirms your catalog and data are ready.

## Workshop flow

- **01 - Setup Check** — confirm workspace, catalog, and shared data
- **L100 — SQL AI Functions** — call AI from pure SQL (classify, extract, parse, summarize, mask, forecast)
- **L100 — Agent Bricks Types** — build the no-code agents in the UI
- **L100 — Agent Evaluation** — score an agent with MLflow and an LLM judge
- **L100 — Short-Term Memory** — give an agent memory on Lakebase
- **L200 / L300** — capabilities and the multi-domain supervisor (added as the workshop grows)
- **90 - Optional Advanced** — fork a hackathon track and build your own copilot
- **99 - Wrap Up** — recap and where to go next

## If something looks broken

- Reattach compute if needed.
- Contact the workshop host.

## End of lab

- Save anything you need before the session ends.
- Do **not** click **End Lab** until you are done — lab resources are removed on end.
